# Pandoc GUI

This software aims to provide a clean and modern GUI to generate PDF documents from markdown using Pandoc, with the ability to pass several options, like : 

- Custom code highlithing theme
- Headings numbers
- Font
- Link color
- Margin
- PDF ENgine
- Table of contents
